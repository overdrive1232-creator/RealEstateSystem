# Real Estate System (C Project)

## Description
A console-based real estate management system developed in C.
It allows users to manage properties, users, and transactions.

## Features
- Add and manage users
- Manage property listings
- Store data using CSV/TXT files

## Files
- main.c → entry point
- admin.c → admin functions
- user.c → user operations
- config.c / config.h → configurations

## How to Compile
gcc main.c admin.c user.c config.c -o app

## How to Run
./app

## Notes
Sample data is included in CSV/TXT files.

## Project Structure
- src/ → source code
- include/ → header files
- data/ → sample data files
- docs/ → documentation
